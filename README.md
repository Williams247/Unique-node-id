
 ## unique-node-id
 A node module for unique ids

```npm
npm i unique-node-id
```

 ```js
const uniqueNodeId = require('unique-node-id');

const data = [
    {   
        id: uniqueNodeId(),
        name: 'Lionel Messi',
        email: 'messi@gmail.com'
    },
    {   
        id: uniqueNodeId(),
        name: 'Sergio Arguero',
        email: 'sergio@gmail.com'
    },
    {   
        id: uniqueNodeId(),
        name: 'Memphis Depay',
        email: 'memphis@gmail.com'
    }
];

console.log(data);
````
